Crie o design de um sistema web responsivo para gestão e visualização de dados geoespaciais e administrativos, com foco em organização, clareza visual e produtividade do usuário. O sistema deve possuir interface moderna, limpa e profissional, voltada para uso institucional.

O sistema deve conter:

Tela de Login

Campos de e-mail e senha

Botão de acesso principal

Opção de recuperação de senha

Layout centralizado, minimalista, com identidade visual institucional

Dashboard Principal

Barra lateral fixa com menu de navegação (Dashboard, Cadastros, Mapa Interativo, Relatórios, Configurações)

Header superior com nome do sistema, usuário logado e botão de logout

Cards com métricas resumidas (total de registros, usuários ativos, dados recentes, indicadores)

Gráficos estatísticos (barra e linha) para visualização de dados

Tela de Mapa Interativo

Área principal com mapa em destaque

Painel lateral para filtros e pesquisa

Lista de resultados vinculada ao mapa

Botões de ativação/desativação de camadas

Modal para exibição detalhada de informações ao selecionar um item

Tela de Cadastro

Formulários organizados em seções

Campos de texto, seleção, datas e upload de arquivos

Botões de salvar, cancelar e editar

Feedback visual de sucesso ou erro

Tela de Relatórios

Filtros por período e categoria

Tabela com paginação

Opção visual para exportação (PDF/Excel)

Indicadores resumidos no topo da página

Estilo visual desejado:

Design moderno e profissional

Paleta de cores baseada em azul institucional e tons neutros (cinza claro e branco)

Uso consistente de espaçamentos e grid

Ícones minimalistas

Componentes reutilizáveis

Layout responsivo (desktop prioritário, com adaptação para tablet)
